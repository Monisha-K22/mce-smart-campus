# MCE Smart Campus – Facility Booking and Timetable Management System

## Project Overview
MCE Smart Campus is a web-based application developed to automate classroom and laboratory booking within the college campus.

The system allows:
- Students to book facilities
- Staff to check availability
- Admin to approve/reject bookings
- Users to view timetable information
- Prevention of timetable conflicts

## Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MySQL

## Features
- User Authentication
- Facility Booking
- Timetable Management
- Booking Approval System
- Conflict Detection
- Auto Expiry of Past Bookings
- Admin Dashboard
- User Management

## Database Tables
- users
- facilities
- timetable
- bookings
